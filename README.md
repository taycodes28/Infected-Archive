My project is a website dedicated to one of my favorite books. It is a guide into the world of "One Dark Window", including characters, places, and magic descriptions. I made this to grow in my expierience with coding. I made this project with HTML and CSS, as well as a little bit of javascript so that I could add a spotify player. When I first started I struggled finding the right color theme and vibe that I wanted for the project. I learned how to get pictures/cards into a grid using CSS.
